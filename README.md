# Student Check-in App

A beginner-friendly Java Swing + SQLite application that allows students to
anonymously share how they are doing in college.

## Features
- Java Swing GUI
- SQLite database
- Anonymous responses
- Conditional questions (projects, internship, etc.)

## Tech Stack
- Java
- Swing
- SQLite
- JDBC
